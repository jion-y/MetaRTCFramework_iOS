## Installation

### 本仓库依赖 lfs,需要安装 git lfs
`brew install git-lfs`

MetaRTCFramework is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'MetaRTCFramework'
```
